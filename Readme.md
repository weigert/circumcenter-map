# CircumcenterFractal

A circumcenter fractal explorer on the GPU.

## Utilization

Execute using

    ./main [windowsizex] [windowsizey]

By default, the window size is 800x800.

## Controls

    - WASD: Move
    - Scroll: Zoom
    - ESC: Toggle Menu

## Compilation

1. Install TinyEngine (On Linux)
2. Use the Makefile (`make all`)
